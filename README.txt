# Support Ticket Classification & Prioritization

## Project Overview

This project automatically classifies customer support tickets and assigns priority levels using Machine Learning and Natural Language Processing (NLP).

## Tools Used

* Python
* Pandas
* Scikit-learn
* TF-IDF Vectorizer

## Features

* Text preprocessing
* Ticket classification
* Priority tagging
* Model evaluation

## Business Benefits

* Faster ticket handling
* Better prioritization of urgent issues
* Improved customer support efficiency
* Reduced manual effort

## Output

The system predicts the ticket category and assigns a priority level (High, Medium, or Low).
